# Pow(x, n)

Implement [pow(x, n)](http://www.cplusplus.com/reference/valarray/pow/), which calculates _x_ raised to the power n (x^n).

__Example 1:__

```
Input: 2.00000, 10
Output: 1024.00000
```

__Example 2:__

```
Input: 2.10000, 3
Output: 9.26100
```

__Example 3:__

```
Input: 2.00000, -2
Output: 0.25000
Explanation: 2^-2 = 1/2^2 = 1/4 = 0.25
```

__Note:__

- -100.0 < _x_ < 100.0
- _n_ is a 32-bit signed integer, within the range [−2^31, 2^31 − 1]
