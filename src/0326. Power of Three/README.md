# Power of Three

Given an integer, write a function to determine if it is a power of three.

**Example 1:**

```pseudo
Input: 27
Output: true
```

**Example 2:**

```pseudo
Input: 0
Output: false
```

**Example 3:**

```pseudo
Input: 9
Output: true
```

**Example 4:**

```pseudo
Input: 45
Output: false
```

**Follow Up:**

Could you do it without using any loop / recursion?
