# Power of Three

Given an integer, write a function to determine if it is a power of three.

__Example 1:__

```pseudo
Input: 27
Output: true
```

__Example 2:__

```pseudo
Input: 0
Output: false
```

__Example 3:__

```pseudo
Input: 9
Output: true
```

__Example 4:__

```pseudo
Input: 45
Output: false
```

__Follow Up:__

Could you do it without using any loop / recursion?
