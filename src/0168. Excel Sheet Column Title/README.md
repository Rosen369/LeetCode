# Excel Sheet Column Title

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:

```pseudo
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB
    ...
```

**Example 1:**

```pseudo
Input: 1
Output: "A"
```

**Example 2:**

```pseudo
Input: 28
Output: "AB"
```

**Example 3:**

```pseudo
Input: 701
Output: "ZY"
```
