# Perfect Squares

Given a positive integer n, find the least number of perfect square numbers (for example, `1, 4, 9, 16, ...`) which sum to n.

__Example 1:__

```pseudo
Input: n = 12
Output: 3
Explanation: 12 = 4 + 4 + 4.
```

__Example 2:__

```pseudo
Input: n = 13
Output: 2
Explanation: 13 = 4 + 9.
```
