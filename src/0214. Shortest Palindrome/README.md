# Shortest Palindrome

Given a string __s__, you are allowed to convert it to a palindrome by adding characters in front of it. Find and return the shortest palindrome you can find by performing this transformation.

__Example 1:__

```pseudo
Input: "aacecaaa"
Output: "aaacecaaa"

```

__Example 2:__

```pseudo
Input: "abcd"
Output: "dcbabcd"
```
