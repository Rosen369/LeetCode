# Reverse Vowels of a String

Write a function that takes a string as input and reverse only the vowels of a string.

__Example 1:__

```pseudo
Input: "hello"
Output: "holle"
```

__Example 2:__

```pseudo
Input: "leetcode"
Output: "leotcede"
```

__Note:__
The vowels does not include the letter "y".
