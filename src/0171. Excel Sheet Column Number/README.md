# Excel Sheet Column Number

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:

```pseudo
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28
    ...
```

__Example 1:__

```pseudo
Input: "A"
Output: 1
```

__Example 2:__

```pseudo
Input: "AB"
Output: 28
```

__Example 3:__

```pseudo
Input: "ZY"
Output: 701
```
