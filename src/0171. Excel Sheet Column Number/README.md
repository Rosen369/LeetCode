# Excel Sheet Column Number

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:

```pseudo
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28
    ...
```

**Example 1:**

```pseudo
Input: "A"
Output: 1
```

**Example 2:**

```pseudo
Input: "AB"
Output: 28
```

**Example 3:**

```pseudo
Input: "ZY"
Output: 701
```
